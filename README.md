# Movie Information System

## Project Overview

Movie Information System is a web-based movie management and review platform that allows users to browse movies by category, view rankings, and access detailed movie information.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Features

### Movie Classification

* Animation Movies
* Science Fiction Movies
* Mystery Movies
* Popular Rankings

### Information Management

* Movie Detail Pages
* Category Navigation
* Ranking Lists

### User Functions

* Registration Interface
* Content Browsing
* Information Retrieval

---

## Skills Demonstrated

* Frontend Development
* Information Management Systems
* Content Organisation
* User Interface Design
* Web Application Development

---

## Project Outcome

Successfully developed a movie information platform capable of organising and presenting large amounts of movie-related content through a structured interface.
